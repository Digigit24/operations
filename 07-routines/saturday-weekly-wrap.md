# ROUTINE: Saturday Weekly Wrap
🔵 ATTACH-TO-CLAUDE — Scheduled routine for the PM Command Center. Fires Saturday at 5:00 PM. Delivered to PM via WhatsApp/Notion.

> End the week with reflection. What got done? What slipped? What do the agents and the team need to do before Monday? The wrap should close loops — not open new ones.

---

## Trigger

Scheduled: Saturday, 5:00 PM
Delivers to: PM's WhatsApp + PM's Notion Command Center page

---

## Inputs

1. All tasks completed this week (Status = Done, this PM, this week)
2. All tasks that were P0/P1 but did NOT complete this week
3. Audit Agent Friday data pull (ad metrics, rank snapshot, social performance this week)
4. Any new client-state.md updates from the week
5. Content calendar: was everything that should have posted actually posted this week?
